# dataType

## `config.dataType`的合法值

| 值   | 说明                                                       | 全平台兼容 |
| :--- | :--------------------------------------------------------- | :--------- |
| json | 返回的数据为 JSON，返回后会对返回的数据进行一次 JSON.parse | 是         |
| 其他 | 不对返回的内容进行 JSON.parse                              | 是         |
