# method

## `config.method`的合法值

可以使用大写，也可以使用小写。

| 值      | 说明 | 全平台兼容 |
| :------ | :--- | :--------- |
| OPTIONS |      |            |
| GET     |      | 是         |
| HEAD    |      |            |
| POST    |      | 是         |
| PUT     |      | 是         |
| DELETE  |      | 是         |
| TRACE   |      |            |
| CONNECT |      |            |
